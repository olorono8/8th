1. What is an Exponent?

An exponent tells us how many times a number is multiplied by itself.

Example: 2³ means 2 × 2 × 2 = 8

Example:

5² = 5 × 5 = 25

3⁴ = 3 × 3 × 3 × 3 = 81

2. Parts of a Power

  • Base: The number being multiplied (2 in 2³)
  • Exponent (Power): How many times the base is multiplied (3 in 2³)

Rules of Exponents

Rule 1: Multiplying Powers with Same Base

am × an = am+n

Example: 2² × 2³ = 22+3 = 2⁵ = 32

Rule 2: Dividing Powers with Same Base

am ÷ an = am-n

Example: 5⁴ ÷ 5² = 54-2 = 5² = 25

Rule 3: Power of a Power

(am)n = am×n

Example: (3²)³ = 32×3 = 3⁶ = 729

Rule 4: Power of a Product

(ab)n = an × bn

Example: (2×3)² = 2² × 3² = 4 × 9 = 36

Rule 5: Power of a Fraction

(a/b)n = an / bn

Example: (2/3)² = 2² / 3² = 4/9

Rule 6: Zero Exponent

a⁰ = 1 (a ≠ 0)

Example: 7⁰ = 1

Rule 7: Negative Exponent

a⁻ⁿ = 1 / aⁿ

Example: 2⁻³ = 1 / 2³ = 1/8

What is a⁰ equal to?

a⁰ = 1 (for any a ≠ 0)

Quick Examples

  • 4³ = 64
  • 10² = 100
  • (2 × 5)² = 2² × 5² = 100
  • 3⁰ = 1
  • 5⁻² = 1/25
  • 7³ = 343
  • 2⁴ × 2² = 2⁶ = 64
  • (4²)² = 4⁴ = 256
